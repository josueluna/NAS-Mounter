NAS Mounter

Version 0.1 · Beta

A minimal macOS utility to quickly mount SMB network drives using saved credentials and a clean, modern interface.

View on GitHub Download

🚀 What does it do?

NAS Mounter simplifies connecting to SMB network drives by handling authentication and mounting in one smooth flow — no Finder popups, no repeated logins.

⚙️ How it works

  1. Enter NAS IP or host
  2. Provide credentials
  3. (Optional) Browse available shares
  4. Click Connect
  5. Drive mounts instantly

✨ Features

  • 🔗 One-click SMB mounting
  • 🔐 Secure credential storage via macOS Keychain
  • 📁 Browse available shares before connecting
  • ⚡ Fast connection with no system popups
  • 🧠 Remembers your credentials
  • 🧩 Clean, minimal macOS UI

🔐 Security

  • Credentials stored securely in Keychain
  • No external servers or tracking
  • Local-only execution

🛠 Requirements

  • macOS
  • Network access to SMB-enabled NAS

🧪 Status

This project is currently in Beta. Core functionality is stable, but improvements and refinements are ongoing.

🛠 Troubleshooting
