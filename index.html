macOS menu bar app for SMB / NAS shares

Mount your NAS. Right from the menu bar.

NAS Mountie is a lightweight macOS app for mounting SMB/NAS shares quickly, safely, and with as little friction as possible. It remembers connection profiles by Wi-Fi network, stores credentials securely in the macOS Keychain, and can automatically mount saved shares when your Mac reconnects to a known network.

Core features

Small, focused, and built for less friction.

NAS Mountie is intentionally narrow in scope: fast SMB mounting, network-aware memory, secure credential handling, and a compact macOS-native experience that stays out of the way.

⟲

Quick SMB mounting

Mount SMB shares from a NAS using an IP address or hostname, then select one or more shares to mount without extra clutter.

⌘

Menu bar first

A compact menu bar popover keeps the workflow fast and focused, without turning NAS access into a full-size utility app.

🔐

macOS Keychain

Credentials are stored securely using the macOS Keychain. Passwords are not kept as plain text in source code or app files.

◌

Per Wi-Fi profiles

NAS Mountie remembers connection profiles by Wi-Fi network so your home NAS profile stays separate from other environments.

↗

Auto-mount on known networks

When your Mac reconnects to a familiar Wi-Fi network, saved shares can mount automatically with much less repetitive setup.

⚙

Useful macOS settings

Includes support for Launch at Login, optional Dock icon visibility, and a lightweight UI tailored to everyday local use.

How it works

From menu bar click to mounted share in a few steps.

The workflow is intentionally direct: enter your NAS details, browse shares, pick what you need, and mount. If you want, NAS Mountie remembers the setup for the current Wi-Fi network.

1

Open from the menu bar

Launch NAS Mountie from the macOS menu bar instead of jumping into a large utility window.

2

Enter NAS credentials

Type the NAS address, username, and password. You can optionally enable Remember Password.

3

Browse and select shares

View available SMB shares from the connected NAS and choose one or multiple shares to mount.

4

Mount and move on

Click Mount. NAS Mountie can save the connection profile for the current Wi-Fi network and mount again automatically next time.

Network profiles

One app, different networks, less accidental crossover.

NAS Mountie remembers connection profiles by Wi-Fi network. That means it can behave intelligently at home, while staying empty or separate on another network.

⌂

What a profile can remember

  • Wi-Fi network name
  • NAS host or IP address
  • Username
  • Previously mounted shares
◎

Why it matters

  • At home, it can remember your home NAS and selected shares.
  • On another network, it starts clean unless you save a separate profile there.
  • This helps avoid showing or mounting the wrong shares in the wrong place.
  • Passwords are handled separately through the macOS Keychain.
Security & requirements

Secure where it counts. Practical where it matters.

NAS Mountie uses the macOS Keychain for secure credential storage and is designed for modern macOS workflows, including features like Launch at Login.

🔒

Security

Credentials are stored in the macOS Keychain. Passwords are not stored as plain text in app files or source code.



Requirements

  • macOS 13 or later recommended
  • SMB sharing enabled on your NAS
  • Network access to the NAS
  • Xcode for building from source

Installation from source

NAS Mountie is currently in beta. There is no App Store build or GitHub release yet, so the current path is to build it locally from source. 

git clone https://github.com/josueluna/NAS-Mountie.git
open NAS-Mountie/NAS-Mountie.xcodeproj
Build and run:
Cmd + R
Create a local usable .app:
1. Set scheme to Release
2. Build the project
3. Product > Show Build Folder in Finder
4. Open Products/Release
5. Copy NAS-Mountie.app to /Applications
